Requirements

    PHP >=7.2
   

Supported Image Libraries

    GD Library (>=2.0)
    Imagick PHP extension (>=6.5.7)


## project setup

## 1) composer update

## 2) setup a database through .env file

## 2) php artisan migrate

## 4) php artisan serve --port 8000

   N.B (port should be 8000) because of api endpoint)

that's it.
